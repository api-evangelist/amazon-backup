# Amazon Backup API

AWS Backup is a fully managed backup service that centralizes and automates the backup of data across AWS services, enabling you to configure backup policies, monitor backup activity, and restore resources with a single, unified console and API.

## APIs

- **Amazon Backup API** - API for centrally managing and automating backups across AWS services including Amazon EBS, Amazon RDS, Amazon DynamoDB, Amazon EFS, Amazon FSx, Amazon EC2, and AWS Storage Gateway. Supports creating backup plans, managing backup vaults, starting and monitoring backup jobs, and restoring resources.
